# Plan 6.4: Referral Dashboard & Analytics

## Objective
Provide a UI for organizers to generate referral links and view analytics (clicks, conversions).

## Context
- .gsd/phases/6/SPEC.md

## Tasks

<task type="auto">
  <name>Referral Analytics API</name>
  <files>
    app/api/organizer/events/[id]/referrals/route.ts
    app/api/referral/[code]/click/route.ts
  </files>
  <action>
    - Create organizer API `GET /api/organizer/events/[id]/referrals` to fetch all referral links and their stats.
